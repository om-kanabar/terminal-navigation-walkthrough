1. My labs folder is in the root of my user (`~`)
1. The tilda (`~`) is shorthand for the root of your user, so `/Users/your-user`
1. The difference between `.` and `..` is that `.` refers to your current directory wheras `..` refers to the directory a step up.
1. An absolute path for my lab folder is `/Users/Om/labs`
1. A relative path for my lab folder is `~/labs`
1. My hidden file demonstrated that with `ls` it does not show, but with `ls -la` it showed. 
1. I redirected `pwd` into a file with `pwd > lab-location.txt'
